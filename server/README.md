## Setup
Cross over to this directory
```bash
cd OpenAYEye/server
```

In this directory, first create and install conda env,
```bash
conda env create -f environment.yml
```

Then enter conda environment
```bash
. activate flaskdev
```

## Testing & Development
Run
```bash
fastapi dev app.py
```
